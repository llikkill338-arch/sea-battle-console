# Морской бой — Консольный Бой

Консольная игра «Морской бой» в стилистике ретро 80-х годов.
Разработана на C++ с использованием WinAPI.

## Требования

- Windows 7/8/10/11
- MinGW-w64 (g++) или MSVC
- Консоль с поддержкой UTF-8 и псевдографики

## Компиляция

### Через Makefile (MinGW)

```bash
make
```

### Вручную

```bash
g++ -Wall -Wextra -std=c++11 -O2 -static -o SeaBattle.exe \
    src/main.cpp src/utils.cpp src/graphics.cpp \
    src/game_logic.cpp src/bot_ai.cpp src/screens.cpp
```

## Запуск

```bash
SeaBattle.exe
```

## Управление

### В меню
- **Стрелки ВВЕРХ/ВНИЗ** — выбор пункта
- **ENTER** — подтвердить
- **ESC** — выход

### Расстановка кораблей
- **Стрелки** — движение курсора
- **ПРОБЕЛ** — поворот корабля (горизонталь/вертикаль)
- **ENTER** — установить корабль
- **ESC** — выход

### Бой (против бота / 1 на 1 по очереди)
- **Стрелки** — движение прицела
- **ENTER** — выстрел
- **ESC** — выход

### Бой (разделённый экран)
- **Игрок 1:** W, A, S, D + Left SHIFT (выстрел)
- **Игрок 2:** Стрелки + ENTER (выстрел)
- **ESC** — выход

## Режимы игры

1. **Бой с ботом** — игра против компьютера (лёгкий / сложный ИИ)
2. **Бой 1 на 1 (по очереди)** — два игрока на одном компьютере по очереди
3. **Бой 1 на 1 (разделённый экран)** — оба поля на одном экране
4. **Настройки звука** — включение/выключение звуковых эффектов
5. **Об игре** — правила и управление

## Особенности

- Ретро-стилистика 80-х: неоновые цвета, пиксель-арт, ASCII-графика
- Анимации: пульсация воды, мигающий прицел, взрывы, затухание промахов
- Звуковые эффекты через системный динамик (Beep)
- Умный ИИ в сложном режиме (добивание раненых кораблей)
- Псевдографические рамки и разделители

## Структура проекта

```
src/
├── main.cpp          # Точка входа, главное меню
├── utils.cpp/.h      # Утилиты (звук, глобальные переменные)
├── graphics.cpp/.h   # Графика (цвета, курсор, рамки, анимации)
├── game_logic.cpp/.h # Логика игры (поля, корабли, проверки)
├── bot_ai.cpp/.h     # ИИ бота (лёгкий + сложный)
└── screens.cpp/.h    # Экраны и игровые режимы
```

## Технические детали

- Поле 10×10 клеток
- Флот: 1×4-пал, 2×3-пал, 3×2-пал, 4×1-пал (10 кораблей)
- Строгое соблюдение правила: корабли не касаются друг друга даже углами
- При попадании — дополнительный ход
- Микро-MVC архитектура: разделение логики, графики и управления
